Standalone demo UI

Disaster Tracker Dashboard

Explore active disasters, ZIP-code impact, resource signals, and source health from a UI that is fully isolated from the ASP.NET application.

Connection

Point the demo at any compatible API instance.

Default local API: http://localhost:5096

Overview

Quick summary of the current snapshot.

Active disasters

Filter the normalized live feed and inspect current incidents.

Event detail

Click an active event to inspect the normalized record.

ZIP impact lookup

Check whether a ZIP code is currently impacted.

Source health

Operational status and data freshness by feed.

Resource impacts

Explainable strategic-region signals triggered by active events.

Demo UI only. This interface is intentionally separate from the ASP.NET codebase and talks to the API over HTTP.
